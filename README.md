# vuejs-iota-cookbook

This is a collection of different vue applications to learn how to code with IOTA and Vue.js. 

## Project setup
```
git clone https://github.com/einfachiota/vuejs-iota-cookbook.git
cd vuejs-iota-cookbook
```
